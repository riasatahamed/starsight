Data Sources & Attribution

Star PositionsGaia DR3 — ESA's billion-star catalog. Positions accurate to microarcseconds for bright stars. Filtered for G < 6.5 mag with high astrometric quality.
Deep Sky ObjectsOpenNGC — License-friendly NGC/IC database compiled from professional sources. Includes positions, magnitudes, and object types for galaxies, nebulae, and clusters.
Planets, Moon & SunAstronomy Engine by Don Cross — VSOP87-based ephemeris accurate to ±1 arcminute. Calculates positions, rise/set times, phases, and illumination for all solar system bodies.
Constellation DataIAU Constellation Boundaries (Delporte 1930) and Stellarium line data. 88 official constellations with accurate boundary polygons.
Weather & LocationOpen-Meteo API — free open-source weather data. No API key required. Licensed under CC BY 4.0.
NASA APODNASA Open APIs — Astronomy Picture of the Day. Public domain images unless noted otherwise.
All astronomical calculations use high-precision data sources. Star positions from Gaia DR3 include proper motion corrections. Planet positions use VSOP87 ephemeris via Astronomy Engine. Deep-sky objects from OpenNGC. Constellation boundaries follow IAU Delporte (1930) definitions. For research-grade astrometry, consult SIMBAD or VizieR.
